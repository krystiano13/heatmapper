# Heatmapper App

## About
Simple Fullstack app, that allows to create and manage heatmaps.

## Tech Stack
* React
* Ruby on Rails
* Typescript
* TailwindCSS

## Setup
* install ruby and rails gem
* install node.js
* in server folder run ```bundle``` and ```rails s``` commands
* in client folder run ```npm i``` and ```npm run dev``` commands
